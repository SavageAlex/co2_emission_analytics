# NodeJS + Express + VueJS + PostgreSQL

## Project setup

## NodeJS Server
Database setup file
```
/app/config/db.config.js
```
Install modules
```
npm ci
```
For run server
```
node server
```

## Frontend VueJS APP
App directory
```
/frontend-co2-analytics/
```
Install modules
```
npm ci
```
For run app
```
npm run serve
```
